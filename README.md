# Marketing Campaign Performance Analysis using SQL

## Problem Statement
Companies invest heavily in marketing campaigns, but not all campaigns generate profit. This project analyzes campaign spend, user acquisition, and revenue to identify high-ROI and underperforming campaigns.

## Dataset Overview
- campaigns: campaign details and channels
- marketing_spend: campaign-wise spending
- users: users acquired via campaigns
- orders: revenue generated by users

## SQL Concepts Used
- SELECT, WHERE
- JOIN
- GROUP BY, HAVING
- Aggregate functions
- CASE WHEN
- Subqueries
- Views

## Key Analysis
- Campaign-level cost analysis
- Revenue and profit calculation
- ROI-based campaign classification

## Business Insights
- Google Ads generated highest ROI
- Instagram campaign was break-even
- Email campaign showed low but positive profit
- Budget reallocation recommended toward high-performing campaigns
